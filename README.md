# 🧠 NEUROAPP

NEUROAPP es una página web diseñada para ofrecer una experiencia moderna e interactiva centrada en una interfaz tipo aplicación, con sistema de acceso y una estructura organizada en HTML, CSS y JavaScript.

El proyecto está preparado para ejecutarse directamente desde el navegador sin necesidad de servidor.

---

## 📌 Características principales

- Interfaz moderna y limpia
- Página principal (`index.html`)
- Página de login (`login.html`)
- Archivos organizados en carpetas
- Diseño organizado para facilitar mantenimiento y escalabilidad
